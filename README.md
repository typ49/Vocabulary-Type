# Vocabulary-Type

## S6->PMP->Project-2

### Project Description

This Project consist in the implementation of an **[any class](./voc/Any.h)** and an **[optinal class](./voc/Optional.h)** who inherit from the first one.

these classes represent the [any type](https://en.cppreference.com/w/cpp/utility/any) and the [optinal type](https://en.cppreference.com/w/cpp/utility/optional) in C++.

The **[any class](./voc/Any.h)** is a class that can hold instances of any type, and the **[optinal class](./voc/Optional.h)** is a class that can hold an instance of a type or nothing.

a test suite for any and optinal are provided in the **[test](./voc/testVocabularyTypes.cc)** class.

### how to run the test suite

in the build directory run the following commands:

```bash
cmake ..
make
./testVocabularyTypes
```
